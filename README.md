## Installation

### Manual Installation

1. Clone this repository
2. Copy the theme folder to your VS Code extensions directory:
   - **Windows**: `%USERPROFILE%\.vscode\extensions`
   - **Mac/Linux**: `~/.vscode/extensions`
3. Reload VS Code
4. Select the theme from `File > Preferences > Color Theme`

## Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| Deep Purple | `#1a1625` | Editor background |
| Pale Pink | `#ffb3e6` | Strings |
| Lavender | `#e0b3ff` | Constants, types |
| Medium Purple | `#c084fc` | Keywords |
| Soft Pink | `#f0c4f0` | Variables |
| Plum | `#dda0dd` | Functions |
